SL_C0de
=======

**SL_C0de** (/lu'make/) is a python library based on the matlab code SL_code. This module allows to calculate GIA over the world but also the sediment isostasy. It includes function to calculate radial symetric auto gravitational earth visco-elastic response. This module also includes the option to resolve the Sea level equation. This library based on the publication of Mitrovica and Austermann ... ajouter des refs. The aim of this library is to be easy to include new sources of load and link to other python library.
